# AI-Powered Diabetes Risk Prediction System

XGBoost-based web application for Type 2 diabetes risk assessment with explainable AI.

## Features
- 97.19% accuracy XGBoost model
- Real-time risk predictions
- Explainable AI (SHAP)
- Dynamic personalized recommendations
- Continuous learning from user feedback
- Professional PDF reports

## Tech Stack
- **Backend:** Python, FastAPI, XGBoost, SHAP
- **Frontend:** React, Vite, Tailwind CSS
- **Database:** SQLite
- **ML:** scikit-learn, pandas, matplotlib

## Quick Start

### Backend
\`\`\`bash
cd backend
pip install -r requirements.txt
python main.py
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

Open http://localhost:5173

## Model Performance
- Accuracy: 97.19%
- MCC: 0.8120
- PR-AUC: 0.8927
- Training data: 100,520 samples from 2 datasets
